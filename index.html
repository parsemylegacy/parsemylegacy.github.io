Parse My Legacy!

Parser for positional files

What's parsemylegacy?

parsemylegacy is a simple java 8 parser for positional files.

Examples

Flat file:

WALTER                        WHITE
JESSE                         PINKMAN
SAUL                          GOODMAN

Pojo:

@Line
public class Person {

    @Column(from = 1, to = 30)
    public String firstname;

    @Column(from = 31, to = 60)
    public String lastname;

}

Parse example:

InputStream input = ParserTest.class.getResourceAsStream("/people.txt");
List<Person> people = parser().from(input).parse(Person.class);
